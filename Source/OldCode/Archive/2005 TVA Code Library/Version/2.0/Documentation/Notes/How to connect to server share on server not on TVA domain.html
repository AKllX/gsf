Subject: How to connect to server share on server not on TVA domain...

 

Basic Steps:

 

Version 1.1 Steps: 

  1. Add a reference to TVA.Shared.dll
  2. Add an imports for TVA.Shared.Identity
  3. Call the ConnectToNetworkShare

 

Version 2.0 Steps: 

  1. Add a reference to Tva.Core.dll
  2. Add an imports for Tva.Identity
  3. Call the ConnectToNetworkShare

 

 

If a box is not on a domain, its name is the domain:

 

For example, the box named SEICCPA is not on the TVA domain and it has a share called OAG_FILELINK_IN.  Using a local username and password on that box , I can access the drive like this:

 

ConnectToNetworkShare(“\\seiccpb\OAG_FILELINK_IN", “telegyr”, “mypassword”, “SEICCPA”)

 

This acts just like going to the command prompt and doing a “NET USE” on the remote drive.

 

Once connected, I can access files on the share, for example:

 

myFile = File.Open(“\\seiccpb\OAG_FILELINK_IN\TestFile.txt")

 

One important thing to note, you must call DisconnectFromNetworkShare when you’re finished.  This is a “global” action, i.e., once connected the logged in user will have access to this share (even though they may not know it) until you disconnect. 

 

 

Production Example:

 

 #If VBC_VER = 8.0 Then

Imports Tva.IO.FilePath

Imports Tva.Identity.Common

#Else

Imports TVA.Shared.FilePath

Imports TVA.Shared.Identity

#End If

 

    Private Class ProxyRule

 

        Public FileSpecs As String()

        Public Destination As String

        Public UserName As String

        Public Password As String

        Public Domain As String

 

        Private Shared impersonationSync As New Object

 

        Public Function ApplyRule(ByVal fileName As String, ByVal rootFileName As String) As Boolean

 

            ' If this file matches the file specs defined for this rule we copy it to the specified location

            If IsFilePatternMatch(FileSpecs, rootFileName) Then

                ' Make sure we synchronize this operation, don't want to step on our own toes where logins are concerned!

                SyncLock impersonationSync

                    Dim destFileName As String = AddPathSuffix(Destination) & rootFileName

                   

                    ' Connect to network share, if specified

                    If Len(UserName) > 0 Then ConnectToNetworkShare(Destination, UserName, Password, Domain)

 

                    If File.Exists(destFileName) Then

                        WaitForWriteLock(destFileName, 15)

                        File.Delete(destFileName)

                    End If

 

                    File.Copy(fileName, destFileName)

 

                    ' Restore original user rights

                    If Len(UserName) > 0 Then DisconnectFromNetworkShare(Destination)

 

                    Return True

                End SyncLock

            End If

 

            Return False

 

        End Function

 

    End Class
